jDiskMark is a disk io utility written in java. 


== Builds == 

https://sourceforge.net/projects/jdiskmark/


== Usage ==

1. Requires java 8.

2. to run:
    
   $ java -jar jDiskMark.jar

   On windows double click executable jar file.
